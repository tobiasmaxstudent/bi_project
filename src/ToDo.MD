## Requirements

### Requirement 1: Upload and Parse CSV File
The user shall be able to upload a CSV file from their computer. The file shall be parsed using the CSV module and made available for viewing in the application.

### Requirement 2: Display Data in Table Format
The parsed CSV data shall be displayed in a table format showing all columns and rows. The table shall be easily readable and properly formatted.

### Requirement 3: Filter and Display Statistics
The user shall be able to filter and view basic statistics about the data, such as finding maximum values or sorting by specific columns.
